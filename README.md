 Snake Game in C

 Descriere
Joc clasic Snake în C pentru terminalul Windows.  
Șarpele se mișcă pe o tablă pătrată, mănâncă mâncare (`F`) și crește.  
Jocul se termină dacă șarpele lovește pereții sau propria coadă.

 Compilare și rulare
1. Compilează codul folosind un compilator C (ex. gcc):

```bash
gcc main.c snake.c -o snake.exe
